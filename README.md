# Vikrant Yadav - Portfolio

A comprehensive portfolio showcasing projects and experience across multiple companies.

## Features

- Company-wise project organization
- Tagged projects for easy filtering
- Responsive design
- Modern UI/UX
- Smooth animations

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## View Portfolio

Open `index.html` in a web browser or deploy to GitHub Pages.

## Structure

- `index.html` - Main portfolio page
- `styles.css` - Styling and responsive design
- `script.js` - Interactive features and animations
